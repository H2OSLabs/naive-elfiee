
## 1   核心逻辑重构：意图 -闭环 -进化

在 Elfiee 中，一个完整的进化循环应该是自动发生的：

1. **意图捕获**：用户/AI 创建 Task。
    
2. **执行闭环**：AI 通过 MCP 操作 Block，并在 Terminal 中验证结果。
    
3. **模式提取 (Evolution)**：当 `task.commit` 成功且测试通过时，系统自动分析该 Task 的关系链。
    
4. **Skill 固化**：系统将该类问题的解决路径压缩进 `.elf/Skills/`，成为下一次 AI 处理类似任务的直觉。

## 2   核心功能优先级
### 2.1   打通闭环


| **功能名称**            | **描述**                                                                                         | **为什么现在做**                      | **优先级** |
| ------------------- | ---------------------------------------------------------------------------------------------- | ------------------------------- | ------- |
| Terminal 报错自动归因     | 当 Terminal 执行报错时，自动将错误堆栈（Error Stack）写回当前 Task Block 的子 Block，并标记为 `block_type: error_report`。 | 为 AI 提供失败的教训，这是 Skill 进化的关键负反馈。 | P0      |
| Contextual Proposal | AI 根据当前 Task 及其 `implement` 关系链，自动生成一系列 `proposed_commands`（如：新建文件、修改代码、运行测试）。                 | 验证 AI 是否能理解现有的因果链。              | P0      |
| Task 状态机联动          | 当 Git 提交成功（`task.commit`）且最后一次 Terminal 运行成功时，Task 自动变为 `Verified` 状态。                         | 建立成功的明确边界，触发 Skill 提取。          | P1      |
|                     |                                                                                                |                                 |         |

### 2.2   从 Event 生成 Skill


| **功能名称**                       | **描述**                                                                | **逻辑与价值**                                    | **优先级** |
| ------------------------------ | --------------------------------------------------------------------- | -------------------------------------------- | ------- |
| Auto-Skill Generator           | 监听 `task.commit` 事件。成功后，AI 自动总结该 Task 的问题-逻辑-方案-验证四元组，并更新 `SKILL.md`。 | 实现 Event 到 Skill 的自动转化                       | P0      |
| Pattern Matcher (技能唤醒)         | 当用户创建新 Task 时，系统搜索历史已闭环的任务 Event 相似度。                                 | 如果有相似路径，自动将旧的成功经验作为参考上下文（Reference）注入给 Agent | P1      |
| Skill 质量评审 (Human-in-the-loop) | 在 GUI 中提供一个列表，展示 AI 自动生成的新技能草稿，由人类确认是否将其从临时经验转为全局 Skill。              | 确保进化的方向不产生偏离，体现人类是决策审核者的原则。                  | P1      |
